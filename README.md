# ExaByte - Music Distribution Platform

A modern, professional website for ExaByte music distribution platform. Built with HTML, CSS, and JavaScript.

## Features

✨ **Modern Design**
- Sleek cyberpunk-inspired UI with gradient accents
- Fully responsive design for all devices
- Smooth animations and transitions

🎵 **Complete Sections**
- Hero section with call-to-action
- Features showcase (6 key features)
- Pricing plans (Starter, Professional, Enterprise)
- About section with statistics
- Contact form
- Navigation and footer

📱 **Responsive**
- Mobile-friendly design
- Tablet optimization
- Desktop experience optimized

🎨 **Interactive**
- Smooth scroll navigation
- Hover animations
- Form validation
- Scroll-triggered animations

## How to Use

1. **Open in Browser**: Simply open `index.html` in your web browser
2. **No Installation Required**: Pure HTML/CSS/JavaScript - no build tools needed
3. **Customize**: Edit the HTML, CSS, and JavaScript to match your branding

## Customization Tips

### Colors
Edit the CSS variables in `assets/styles.css`:
```css
:root {
    --primary: #00d4ff;        /* Main cyan color */
    --secondary: #ff006e;      /* Pink accent */
    --dark: #0a0e27;           /* Dark background */
    --accent: #00f5ff;         /* Bright cyan */
}
```

### Content
- Edit text in `index.html`
- Modify pricing plans and features
- Update company statistics in the About section
- Add your social media links in the footer

### Sections
- **Hero**: Update title and subtitle
- **Features**: Modify the 6 feature cards
- **Pricing**: Adjust plan names and prices
- **About**: Update company information and stats
- **Contact**: Connect to your backend for form submissions
